The Display Property

  • Every element has a default display value depending on what type of element it is.
  • The default for most elements is usually block or inline.
    • A block element is often called a block-level element.
    • An inline element is always just called an inline element.

Block Elements

div tags are the "standard" block level element.

Notice that this div is a block level element. It spans the entire width of the viewport.

Inline

span is the standard inline element. Inline elements wrap text and elements without disrupting the flow of things around it.

Display None

Another common display value is none.

Setting display: none will render the page as though the element does not exist.
